Hojas de Estilo CSS



Conceptualización


Las hojas de estilo en cascada (CSS) ofrecen un modo flexible de agregar estilos a las páginas de su sitio Web. Este conjunto de reglas de formato rige el aspecto de sus páginas y le permite definir las fuentes, los colores, el formato y otras características de presentación.

La separación entre contenido y presentación mejora la flexibilidad, permitiendo a los autores definir un único conjunto de estilos que miles de documentos utilizarán, reduciendo así el tiempo y trabajo que se debe invertir en cada actualización. Asimismo, la implementación de CSS permite a un documento ser exclusivamente semántico, al liberarlo de la necesidad de usar elementos presentacionales.



Breve Historia


Las hojas de estilos aparecieron poco después que el lenguaje de etiquetas SGML, alrededor del año 1970. Su aparición se debe al gran impulso de los lenguajes de hojas de estilos implementados en Internet y el crecimiento exponencial del lenguaje HTML para la creación de documentos electrónicos. El organismo W3C (World Wide Web Consortium), encargado de crear todos los estándares relacionados con la web, propuso la creación de un lenguaje de hojas de estilos específico para el lenguaje HTML, a partir de allí se presentan dos propuestas: CHSS (Cascading HTML Style Sheets) y SSP (Stream-based Style Sheet Proposal).

En 1995, el W3C decidió apostar por el desarrollo y estandarización de CSS y lo añadió a su grupo de trabajo de HTML. Posterior a ello, a principios de 1997, el W3C decide separar los trabajos del grupo de HTML en tres secciones: el grupo de trabajo de HTML, el grupo de trabajo de DOM y el grupo de trabajo de CSS.

Luego de esta separación, el 12 de Mayo de 1998, el grupo de trabajo de CSS publica su recomendación oficial, conocida como "CSS nivel 2", la cual permite que los autores y los usuarios asocien un estilo por ejemplo: fuentes, colores, espaciado, bordes etc a los documentos estructurados HTML y aplicaciones XML.

La versión de CSS que utilizan todos los navegadores de hoy en día es CSS 2.1



Conceptos Principales



Tipos de CSS






La hoja de estilo en cascada (CSS) se usa para establecer el estilo en páginas web que contienen elementos HTML, es decir permite establecer el color de fondo, el tamaño de la fuente, la familia de la fuente, el color, etc. de los elementos en las páginas web.
Se pueden calsificar en:

* CSS en línea: se utiliza para dar estilo a un elemento HTML específico. Para este tipo de estilo CSS, sólo tendrás que añadir el atributo style a cada etiqueta HTML, sin utilizar selectores.

* CSS interno o incrustado: requiere que añadas la etiqueta style en la sección head de tu documento HTML y es un método eficaz para dar estilo a una sola página.

* CSS externo: enlazarás tus páginas web a un archivo .css externo, que puede ser creado con cualquier editor de texto de tu dispositivo. Este tipo de CSS es un método más eficiente, especialmente para estilizar un sitio web grande.



Ventajas de usar CSS



CSS es uno de los lenguajes más practicos del mundo del desarrollo actual, ya que ayudan día a día al programador a realizar un desarrollo llamativo y oroginal. Dentro de las ventajas que se encuentran al implementarlo se encuentra:

  • Mas Accesible: Los CSS permiten modificar el formato de una página web según sean las necesidades del usuario. Por ejemplo, personas con deficiencias visuales.

  • Más optimizados: El ancho de banda necesario para cargar páginas web con CSS es mucho menor que el resto. La hoja de estilo se almacena en cache y no tiene que descargar la página web una y otra vez

  • Multidispositivo: Las hojas de estilo permiten configurar la página para diferentes dispositivos por ejemplo: móviles, tables, lectores de voz.
