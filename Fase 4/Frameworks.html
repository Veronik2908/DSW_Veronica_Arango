Frameworks CSS



Conceptualización


Un framework es un conjunto de herramientas, librerías, convenciones y buenas prácticas que pretenden encapsular las tareas repetitivas en módulos genéricos fácilmente reutilizables. Por su parte, un framework CSS es un conjunto de herramientas, hojas de estilos y buenas prácticas que permiten al diseñador web olvidarse de las tareas repetitivas para centrarse en los elementos únicos de cada diseño en los que puede aportar valor.

Los frameworks CSS más completos incluyen utilidades para que el diseñador no tenga que trabajar en ningún aspecto genérico del diseño web. Por este motivo, es habitual que los mejores frameworks CSS incluyan herramientas para:

  • Neutralizar los estilos por defecto que aplican los navegadores.

  • Manejar correctamente el texto, de forma que todos los contenidos se vean exactamente igual en todos los navegadores y que sean adaptables para mejorar su accesibilidad.

  • Crear cualquier estructura compleja o layout de forma sencilla, con la seguridad de que funciona correctamente en cualquier versión de cualquier navegador.

    Ventajas y Desventajas de los Frameworks de Desarrollo



    Un framework CSS es un programa que ofrece diversas opciones para que empleas en el desarrollo de HTML, potenciando la velocidad de desarrollo y facilitando el porceso de creación de aplicaciones. La verdad es que puede presentar tanto ventajas como desventajas como se puede observar a continuación:

  • Ventajas:

    • 1. Agilización del proceso de desarrollo.
    2. Facilita la funcionalidad en diversos navegadores.
    3. Te permite trabajar con layout simétrica.
    4. Refuerza buenos hábitos de diseño.

  • Desventajas:

    • 1. Limita tu libertad.
    2. Añade código extra.
    3. Te fuerza a usar la semántica propia de la framework.
    4. Podría ser una pérdida de tiempo.
